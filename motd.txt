- 바닐라 게임 플레이 유지
- 세밀한 성능 튜닝으로 쾌적한 서버
- 편의 기능 추가
- 버그 수정 및 억까 방지
- 낭비되는 시간 절약

[ 기능 ]
- 핑 찍기        → 보컬라이저 'Look'
- 아이템 전달    → 밀치기 또는 R키
- 아이템 던지기  → 바인드 설정 후 키 입력
- 현재 진행도    → !p
- 하드모드 확인  → !hardi
- 탄약 배수 확인 → !ammi
- MVP 확인       → !mvp, !mvpme
