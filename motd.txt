[ 기능 ]
- 핑 찍기        → 보컬라이저 'Look'
- 아이템 전달    → 밀치기 또는 R키
- 아이템 던지기  → /drop
- 자살           → !kill
- 현재 진행도    → !p
- 하드모드 확인  → !hardi
- 탄약 배수 확인 → !ammi
- MVP 확인       → !mvp, !mvpme

  · 가장 뒤에 있는 생존자 위치에서 리스폰
- 봇 자동 정리
  · 생존자가 은신처(안전지대)를 벗어나면 봇 킬
  · 플레이어가 서버에서 나가면 해당 생존자 킬
